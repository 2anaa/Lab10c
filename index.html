Welcome to My Web Page with Materialize

Framework Installation

I installed Materialize via CDN by adding the following links in the `` section: 

            <link href="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/css/materialize.min.css" rel="stylesheet">
            <link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.3/css/all.min.css" rel="stylesheet">

Building the Web Page

I used Materialize's grid system and CSS classes to build the page. For example, I used the `container` class for layout and Materialize's typography for headings and text.

Difficulties Encountered and Solutions

  • I initially faced difficulty using Materialize's grid system because I wasn't familiar with the grid structure. I solved this by referring to the Materialize documentation and examples.
  • Setting up Materialize through CDN was a bit tricky at first, as I wasn't sure how to properly link the stylesheets. I followed Materialize's getting started guide and got it working in no time.
